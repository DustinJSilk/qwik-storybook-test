# Qwik Storybook test ⚡️

Setup and install

`pnpm install`

Run development

`pnpm storybook`

Build production

`pnpm storybook.build`

Then to preview the production build, try:

`cd storybook-static && http-server`

You might need to install http-server to preview the build, `npm install -g http-server`
